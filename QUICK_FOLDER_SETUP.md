### Step 1: Get Your Folder ID from Google Drive

1. **Go to Google Drive:**
   - Open: https://drive.google.com
   - Make sure you're signed in

2. **Create or Find Your Folder:**
   - Create a new folder called "Mail Reports" (or use existing)
   - **Double-click to OPEN the folder** (important - you must be inside the folder)

3. **Copy the Folder ID from URL:**
   - Look at your browser's address bar
   - You'll see a URL like:

     ```
     https://drive.google.com/drive/folders/1a2b3c4d5e6f7g8h9i0j1k2l3m4n5o6p
     ```
   - The **Folder ID** is the long string after `/folders/`
   - Copy everything after `/folders/` (the entire long string)
   - Example: `1a2b3c4d5e6f7g8h9i0j1k2l3m4n5o6p`

### Step 2: Paste Folder ID in Your App

1. **In the Settings modal**, click: **"Set Folder ID"** button (grey button)

2. **When prompted "Enter Google Drive Folder ID:"**
   - Paste your Folder ID (the long string you copied)
   - Click OK

3. **When prompted "Enter folder name (optional):"**
   - Type: `Mail Reports` (or any name you like)
   - Click OK

4. **You should see:** "Base folder set: Mail Reports" âœ…

5. **Click "Done"** to close Settings
