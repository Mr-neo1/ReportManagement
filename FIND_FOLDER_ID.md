# üìÅ How to Find Google Drive Folder ID

## Method 1: Using Google Drive URL (Easiest)

### Steps:

1. **Go to Google Drive:** https://drive.google.com

2. **Create or navigate to your folder:**
   - Create a new folder (e.g., "Mail Reports")
   - OR open an existing folder you want to use

3. **Open the folder** (double-click to open it)

4. **Look at the URL in your browser:**
   - The URL will look like:
     ```
     https://drive.google.com/drive/folders/1a2b3c4d5e6f7g8h9i0j
     ```
   - The **Folder ID** is the part after `/folders/`
   - In this example: `1a2b3c4d5e6f7g8h9i0j`

5. **Copy the Folder ID** (the long string of letters and numbers)

---

## Method 2: Right-Click Method

1. **Go to Google Drive**
2. **Right-click on the folder** you want to use
3. **Click "Share"** or **"Get link"**
4. **In the sharing dialog, copy the link**
5. **The Folder ID is in the URL:**
   ```
   https://drive.google.com/drive/folders/FOLDER_ID_HERE?usp=sharing
   ```

---

## Method 3: Using Folder Properties

1. **Open the folder in Google Drive**
2. **Click the folder name** at the top
3. **Right-click ‚Üí "Open in new tab"**
4. **Check the URL** - the Folder ID is after `/folders/`

## üîß Using Folder ID in the App:

1. **Click "Set Folder ID" button** in Settings
2. **Paste the Folder ID** (just the ID, not the full URL)
3. **Enter folder name** (e.g., "Mail Reports")
4. **Click OK**

That's it! Your folder is now set up! ‚úÖ

---

## üí° Tips:

- **Folder ID format:** Usually a long string of letters and numbers (like `1a2b3c4d...`)
- **No spaces:** Just the ID, nothing else
- **Case sensitive:** Copy it exactly as shown
- **Works immediately:** Once set, reports will upload to this folder
